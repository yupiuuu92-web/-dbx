ИНСТРУКЦИЯ ДЛЯ RAILWAY:
1. Создай ПРИВАТНЫЙ репозиторий на GitHub.
2. Загрузи туда ВСЕ файлы из этого архива.
3. В Railway выбери 'New Project' -> 'Deploy from GitHub'.
4. В разделе 'Variables' (Переменные) обязательно добавь:
   - API_TOKEN : Твой токен от @BotFather
   - TRACKER_URL : Ссылка на твой проект Railway (будет доступна после деплоя)
